## Project Details

**Project Title:**  
EcoVisionAI - Waste Segregation

**Project Description:**  
Accessibility-first waste segregation · Powered by Gemma 2 via Ollama
"39 million blind people worldwide deserve independence in managing their own waste."

EcoVision AI is a real-time waste detection system built specifically for visually impaired users. Point your webcam at any waste item — the app identifies it, categorises it, and speaks guidance aloud powered by Gemma 2 running locally via Ollama. No cloud. No internet required.

**Tech Stack Used:**  
Frontend -	           React.js -	                      Component-based, excellent accessibility support
Backend -	             FastAPI (Python) -	              Async, WebSocket support, auto API docs
Object Detection -	   YOLOv8n -	                      Fastest model, CPU-capable, real-time
Language Model -	     Gemma 2:2b via Ollama -	        Local, offline, low-RAM, free forever
TTS -	                 Azure Cognitive / pyttsx3 -	    Natural voices with offline fallback
Database -	           Supabase PostgreSQL -	          Free tier, built-in auth, Row Level Security
Hosting -	             Azure VM (Student) + Vercel -	  Free credits, 24/7 uptime
